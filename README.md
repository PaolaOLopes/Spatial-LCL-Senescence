# Cellular Senescence Integrates Inflammatory and Cytotoxic Axes in the Pathological Landscape of Cutaneous Leishmaniasis

Spatial transcriptomics analysis pipeline investigating the spatial organization of cellular senescence in localized cutaneous leishmaniasis (LCL) caused by *Leishmania braziliensis*.

## Main Analyses
- Seurat-based spatial transcriptomics analysis
- Cell2location deconvolution integration
- UCell senescence and cytotoxicity scoring
- Gaussian Mixture Modeling (GMM) for senescence classification
- Differential expression analysis
- GSEA enrichment analysis
- Spatial density mapping of senescent regions
